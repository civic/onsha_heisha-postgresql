#### 4.2.9 型キャスト

- 暗黙に型付けされる3つの定数
    - 文字列 'xxx'
    - ビット文字列 B'xxx'
    - 数値 999
- 他の型の定数は文字列からキャストして使う
    - type 'string'
    - 'string'::type
    - CAST('string' AS type)
    - テーブルの列への挿入時など明示的な場合は省略可能
